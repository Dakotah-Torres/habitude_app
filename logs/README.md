# Habitude — Sprint Handoff Logs

Permanent archive of `truth/handoff.md` at the close of each sprint.

The PM copies the full handoff thread here before clearing `truth/handoff.md`
for the next sprint. `truth/journal.md` is the one-line activity log;
these files are the full working context for each sprint — every handoff,
fix list, ruling, and verification output, in order.
